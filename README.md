## About

MasterCrypto is a focused landing experience for a serious crypto course in Hebrew — the antidote to a market full of affiliate promos and recycled YouTube takes. The page walks visitors through curriculum, instructor credibility, FAQ and a clean enrollment path. Beginner-friendly and intermediate-respectful.

### Sections

| Section | What it shows |
| --- | --- |
| **Hero** | Course headline, value proposition and primary CTA. |
| **Curriculum** | Modules and learning outcomes at a glance. |
| **Topics** | From blockchain fundamentals to portfolio strategy. |
| **Instructor** | Credibility upfront, not buried in fine print. |
| **FAQ** | Straightforward answers to the questions that block sign-up. |
| **Pricing** | Transparent tiers, no hidden cost. |
| **Enroll** | A clear path to enrolling. |
